# สรุปภาพรวมเว็บไซต์สมัชชาคนจน (Assembly of the Poor)

เว็บไซต์สมัชชาคนจน เป็นแพลตฟอร์มที่ถูกพัฒนาขึ้นเพื่อเป็นกระบอกเสียงและสื่อกลางในการนำเสนอเรื่องราว การรณรงค์ และข่าวสารของเครือข่ายภาคประชาชน โดยมีการออกแบบที่เน้นความเรียบง่าย เข้าถึงง่าย และสะท้อนถึงพลังของคนรากหญ้า

## 1. ข้อมูลทางเทคนิค (Tech Stack)
เว็บไซต์นี้ถูกพัฒนาด้วยเทคโนโลยีสมัยใหม่ เพื่อประสิทธิภาพและความรวดเร็วในการใช้งาน:
*   **Frontend:** Next.js 14 (App Router) และ React 18
*   **Styling:** Tailwind CSS (กำหนดสีตาม Brand Guidelines)
*   **Animation:** Framer Motion (สำหรับการทำแอนิเมชันและเอฟเฟกต์การเคลื่อนไหวต่างๆ)
*   **Backend & CMS:** Headless WordPress (ดึงข้อมูลผ่าน REST API จาก `https://assemblyofthepoor.org/wp-json`) พร้อมรองรับ JWT Authentication และใช้งาน ACF (Advanced Custom Fields)
*   **Typography:** Noto Sans Thai Looped (สำหรับภาษาไทยและภาษาอังกฤษ)

## 2. โครงสร้างหลักและฟีเจอร์ของเว็บไซต์
เว็บไซต์ถูกแบ่งออกเป็นหลายส่วนหลักเพื่อให้ผู้ใช้งานสามารถค้นหาข้อมูลและมีส่วนร่วมได้ง่าย:
*   **หน้าแรก (Home):** นำเสนอแคมเปญหลัก (Hero Section), บทความเจาะลึก (In Focus), ข่าวสารล่าสุด (News), ไทม์ไลน์เหตุการณ์, และลิงก์ด่วน
*   **เกี่ยวกับเรา (About):** ข้อมูลประวัติและจุดประสงค์ของสมัชชาคนจน
*   **งานของเรา (Campaigns):** พื้นที่นำเสนอแคมเปญและการรณรงค์ต่างๆ ที่กำลังดำเนินการอยู่
*   **ห้องสื่อคนจน (Media):** แหล่งรวมสื่อ บทความ และเนื้อหามัลติมีเดีย
*   **ค้นหา (Search):** ระบบค้นหาเนื้อหาภายในเว็บไซต์
*   **การมีส่วนร่วม (Get Involved):** ช่องทางสำหรับผู้ที่ต้องการร่วมสนับสนุนหรือเข้าร่วมกิจกรรม
*   **ติดต่อเรา (Contact):** ข้อมูลสำหรับการติดต่อเครือข่าย
*   **ระบบสองภาษา (Bilingual Support):** รองรับการแสดงผลทั้งภาษาไทย (TH) และภาษาอังกฤษ (EN) ผ่านระบบ Language Context (มีสวิตช์สลับภาษา)

## 3. ระบบการออกแบบ (Design System - SIGNAL 39)
เว็บไซต์ใช้แนวคิด "Universal Cognitive Design System" เพื่อให้เกิดความสบายตาและลดภาระการประมวลผลทางสมองของผู้ใช้งาน (Cognitive Ease) โดยมีเอกลักษณ์ดังนี้:
*   **สีหลัก (Brand Colors):** เน้นสีเขียว (`#009253`) ซึ่งเป็นสีหลักของแบรนด์ สื่อถึงความเติบโตและชีวิต ควบคู่กับสีรองอย่างสีเหลือง ครีม ดำ และขาว
*   **ภาพประกอบ (Illustration):** เน้นภาพวาดลายเส้น (Hand-drawn) ที่เป็นธรรมชาติ สื่อถึงผู้คนหลากหลายกลุ่ม อาชีพ และการรวมพลังของภาคประชาชน (ไม่เน้นรูปทรงเรขาคณิตแข็งๆ)
*   **ลวดลาย (Pattern):** มีการใช้ลายเส้นคน (`pattern-green.svg`) เป็นพื้นหลังแบบโปร่งใสเพื่อสร้างมิติ
*   **ความเข้าถึงง่าย (Accessibility):** เน้นความเปรียบต่างของสีที่ชัดเจน (High Contrast) เพื่อให้อ่านง่ายสำหรับผู้ใช้งานทุกคน ตัดสิ่งรบกวนสายตาออก เช่น เงา ขอบ หรือกราฟิกที่ซับซ้อนเกินไป

## 4. การจัดการข้อมูล (API Integration)
การแสดงผลเนื้อหาบนเว็บเป็นการดึงข้อมูลจากแพลตฟอร์ม WordPress เป็นหลัก โดยใช้ Endpoint API เหล่านี้:
*   `/wp/v2/campaigns` ดึงข้อมูลแคมเปญและงานรณรงค์
*   `/wp/v2/articles` ดึงบทความเจาะลึกและสัมภาษณ์
*   `/wp/v2/posts` ดึงข่าวสารความเคลื่อนไหว
*   `/wp/v2/media` ดึงรูปภาพและมัลติมีเดียที่เกี่ยวข้อง

---

**บทสรุป:** 
เว็บไซต์สมัชชาคนจนไม่ได้เป็นเพียงแค่แหล่งรวบรวมข้อมูลข่าวสาร แต่เป็น "เครื่องมือ" ในการเคลื่อนไหวทางสังคมผ่านสื่อดิจิทัล ถูกสร้างขึ้นบนพื้นฐานของเทคโนโลยีเว็บระดับองค์กร (Next.js + Headless CMS) ที่มีความเสถียร รองรับผู้ใช้งานจำนวนมาก และที่สำคัญคือให้ความสำคัญกับการออกแบบที่เข้าใจมนุษย์และสะท้อนจิตวิญญาณแห่งการต่อสู้ของสมัชชาคนจนอย่างแท้จริง
